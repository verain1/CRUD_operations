We are going to create an application to perform the basic CRUD operations. 

Step 1: create the angular application with ng new <filename> command.
Step 2: Open the file in code and run the command npm install -g json-server to install a fake JSON server to store th data.
Step 3: Import the necessary bootstarp files and paste them in 'index.html'.
Step 4: To run the json server command is: 'json-server --watch db.json'.
